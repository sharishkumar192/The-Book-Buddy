# **The Book Buddy**
To run the website 
1. Install all the dependencies.
```
      pip3 install -r requirements.txt
```

2. Run the following command:
```
      python3 app.py 
```
You're all set! Head to localhost:5000 or enter http://127.0.0.1:5000/ on your browser to check out 'The Book Buddy'.
